# Changelog
## 5.0.0
App update v5.0.0 has been released which contains many bug fixes.

Bug fixes:
 - Fixed an issue where the app may disappear in taskbar mode sometimes
 - Fixed an issue where app is not always on top of everything
 - Fixed an issue where homeroom lunch setting is ignored

Prowler:
 - Added support for real time, removed refresh button


New features:
 - Added a welcome dialog which allows the user to enable taskbar mode or start on startup


Other changes:
 - Use croomssched API instead of MikhailHosting for announcements
 - Created CroomsBellSchedule.Core library for android application development
